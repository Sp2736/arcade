# ARCADE  
### Academic Resource & Career Assist Digital Environment

ARCADE is a **university-focused academic and career guidance platform** designed to centralize verified academic resources, provide structured learning roadmaps, and assist students in identifying skill gaps aligned with career roles.

This project is developed as a **Student Group Project (SGP â€“ Semester 4)** under the domain of **Full Stack Web Development**.

## ğŸ¯ Problem Statement

University students often face:
- scattered academic resources across multiple platforms,
- lack of verified study material,
- unstructured preparation for exams and placements,
- confusion regarding required skills for specific career roles.

ARCADE addresses these issues by providing a **role-based, guided, and faculty-involved system** within a university ecosystem.

## ğŸ§  Core Objectives

- Provide **verified subject-wise academic notes** with faculty approval.
- Enable **structured learning through visual roadmaps**.
- Assist students in **career preparation using skill and role gap analysis**.
- Maintain a **repository of verified resume samples**.
- Offer curated **external resources and practice links**.
- Ensure academic integrity using **role-based access control**.

---

## ğŸ‘¥ User Roles

The system supports four distinct roles:

### 1. Student
- View and download approved notes
- Upload notes for approval
- Follow learning roadmaps
- Select skills and target career roles
- View skill gap analysis
- Access resume samples and external resources

### 2. Faculty
- Upload academic notes
- Review and approve student-uploaded notes
- Suggest or validate learning roadmaps
- Monitor basic student engagement

### 3. HOD
- Review escalated academic content
- Provide final approval when required
- Oversight role with limited daily interaction

### 4. Admin
- Manage users and roles
- Maintain resume repository
- Manage external resources
- Control system-level visibility and data
